# Гайдлайн и пайплайн: Реклама детского питания (0–3 года)

Документ консолидирует требования ФЗ «О рекламе» РФ, гайдлайн по категории «Детское питание (0–3 года)» и описание скриптов генерации баннеров. Пайплайн по структуре аналогичен категориям «Ломбарды», «Доверительное управление», «Методы народной медицины».

---

# ГАЙДЛАЙН: Реклама детского питания (0–3 года)

## 1. Фундаментальные правила (Legal & Compliance)

### 1.1. Обязательные элементы (Must-Have)

На любом рекламном носителе **должны присутствовать**:

1. **Возрастная маркировка:** Чёткое указание возраста, с которого разрешён продукт (например: «0+», «с 6 месяцев», «для детей с 1 года»).
2. **Предупредительная надпись (дисклеймер):**
   - Текст по смыслу: «Для детей с [X] месяцев. Молоко матери — идеальное питание для грудного ребенка. Перед вводом нового продукта в рацион проконсультируйтесь со специалистом» (или утверждённый юристами аналог).
   - Надпись должна быть читаемой, контрастной и занимать заметное место.

### 1.2. Категорические запреты (Red Lines)

**Строго запрещено** использовать в тексте или визуальных образах:

1. **Утверждения о замене ГВ:** Фразы, подразумевающие, что продукт является *полноценным заменителем* женского молока.
2. **Преимущества искусственного вскармливания:** Утверждения, что смесь/пюре лучше, удобнее или полезнее грудного вскармливания.
   - *Примеры стоп-слов:* «Лучше маминого молока», «Полная замена ГВ», «Забудьте о кормлении грудью», «Не требует докорма».

---

## 2. Визуальная стратегия (Visual Identity)

Используется для настройки промптов генерации изображений.

### 2.1. Цветовая палитра

- **Рекомендуется:** Пастельные тона, белый, бежевый, светло-жёлтый. Ассоциации с чистотой и безопасностью.
- **Избегать:** Доминирования розового и красного (вызывают отторжение у ЦА).

### 2.2. Образы и герои

- **Приоритет:** Реалистичные изображения счастливых детей и родителей (взаимодействие «Мама + Ребёнок»).
- **Второй план:** Продуктовая группа (баночки, бутылочки, ложечки, текстура пюре/каши).
- **Сниженный приоритет:** Анимированные/мультяшные персонажи.

### 2.3. Триггеры доверия

- Упор на натуральность (свежие овощи/фрукты рядом с упаковкой).
- При наличии документов — «Рекомендовано специалистами», «Разработано с НИИ Питания» и т.п.

---

## 3. Текстовая стратегия (Copywriting & Tone of Voice)

### 3.1. Тональность

- Заботливая, экспертная, спокойная.

### 3.2. Ключевые ценности (USP)

1. **Польза:** Витамины, нутриенты, сбалансированный состав.
2. **Экологичность:** «Без ГМО», «Без консервантов», «Натуральные ингредиенты».
3. **Безопасность:** Гипоаллергенность, отсутствие соли/сахара (где применимо).

### 3.3. Ключевые слова (Keywords)

- *Существительные:* Детское питание, смесь, пюре, каша, прикорм, с рождения, рост, развитие.
- *Прилагательные:* Адаптированная, гипоаллергенная, натуральная, вкусная, полезная.

# Скрипты пайплайна (по аналогии с ломбардами)

## Созданные файлы

| Файл | Назначение |
|------|------------|
| `baby_food_overlay.py` | Валидатор (гайдлайн), тексты, стили, сценарии фонов, дисклеймеры, список брендов |
| `generate_baby_food_banners.py` | Генерация фонов и баннеров (12 без людей + 9 с людьми, в т.ч. детальные) |
| `baby_food_disclaimer_templates.json` | Шаблоны дисклеймеров с плейсхолдерами {age_marking}, {brand_name}, {contacts} |
| `BABY_FOOD_README.md` | Гайдлайн, чек-лист, описание скриптов |

## Форматы баннеров

- **Квадратный:** 1024×1024  
- **Горизонтальный:** 1200×700  
- **Вертикальный:** 800×1200

## Дисклеймеры и бренды

- **Дисклеймеры:** в `baby_food_overlay.py` — `DISCLAIMERS_BY_AGE` согласованы с возрастными группами (0+, с 4 мес., с 6 мес., с 1 года, с 1.5 лет). `BABY_FOOD_DISCLAIMERS` — объединённый список для обратной совместимости.  
- **Шаблоны:** `baby_food_disclaimer_templates.json` — шаблоны с плейсхолдерами `{age_marking}`, `{brand_name}`, `{contacts}` (по аналогии с ломбардами).  
- **Бренды:** списки `BABY_FOOD_BRANDS_RU` и `BABY_FOOD_BRANDS_EU` (Агуша, ФрутоНяня, Gerber, Nutrilon, NAN, Hipp и др.) для подстановки в шаблоны и справки.  

## Примеры запуска

```bash
# Один баннер по сценарию
python scripts/generate_baby_food_banners.py --scenario jar_puree_lower_right_gradient

# Все сценарии, горизонтальный формат
python scripts/generate_baby_food_banners.py --all-scenarios --format horizontal --count 5

# Только сценарии без людей
python scripts/generate_baby_food_banners.py --without-people --output output/baby_food_no_people

# Только сценарии с людьми (мама + ребёнок)
python scripts/generate_baby_food_banners.py --with-people --output output/baby_food_people

# Только фоны (без текста)
python scripts/generate_baby_food_banners.py --backgrounds-only --format square --all-scenarios

# Несколько сценариев по имени
python scripts/generate_baby_food_banners.py --scenarios "jar_puree_lower_right_gradient,baby_bottle_lower_left_gradient" --count 2

# Список сценариев и требования
python scripts/generate_baby_food_banners.py --list-scenarios
python scripts/generate_baby_food_banners.py --show-requirements

# Проверка текста на соответствие гайдлайну
python scripts/generate_baby_food_banners.py --validate "Лучше маминого молока"
```

## Валидация в коде

- `BabyFoodValidator.check_forbidden(text)` — поиск запрещённых формулировок.  
- `BabyFoodValidator.validate(headline, description, disclaimer, age_marking)` — полная проверка: запреты, наличие фразы о грудном молоке, призыва к консультации специалиста, возрастной маркировки.  
- В `BabyFoodBannerOverlay.apply()` при `validate=True` выполняется эта валидация; при нарушениях выбрасывается `ValueError`.

# ЧЕК-ЛИСТ: Проверка макета перед запуском

Использовать для финальной валидации любого креатива (баннера, видео, лендинга).

## Блок 1: Критические ошибки (Kill-factors)

*Если хотя бы один пункт — «Нет», макет отправляется на переработку.*

| № | Вопрос проверки | Да / Нет |
|---|-----------------|----------|
| 1 | Присутствует ли предупредительная надпись: «Проконсультируйтесь со специалистом...»? | [ ] |
| 2 | Присутствует ли фраза: «Молоко матери — идеальное питание...»? | [ ] |
| 3 | Указан ли возраст (0+, 6 мес+, для детей с 1 года и т.д.)? | [ ] |
| 4 | **Отсутствуют** ли утверждения, что продукт заменяет женское молоко? | [ ] |
| 5 | **Отсутствуют** ли утверждения о преимуществах искусственного вскармливания перед грудным? | [ ] |

## Блок 2: Визуальное соответствие

| № | Вопрос проверки | Да / Нет |
|---|-----------------|----------|
| 6 | Цветовая гамма преимущественно пастельная (белый, бежевый, жёлтый)? | [ ] |
| 7 | Отсутствуют агрессивные красные и розовые цвета в больших объёмах? | [ ] |
| 8 | Изображены счастливые люди (мама/ребёнок) или качественная продуктовая группа? | [ ] |
| 9 | Изображение чёткое, без искажений (особенно лиц и рук)? | [ ] |

## Блок 3: Смысловое наполнение

| № | Вопрос проверки | Да / Нет |
|---|-----------------|----------|
| 10 | Есть ли упоминание пользы (витамины, натуральность, без ГМО)? | [ ] |
| 11 | Слоган краткий и понятный? | [ ] |
| 12 | Бренд считывается однозначно (если применимо)? | [ ] |

# Промпты для генерации изображений (шаблон)

- **Subject:** Happy young mother feeding cute baby with spoon from glass jar, high quality photography.  
- **Environment:** Cozy, bright kitchen, soft morning light.  
- **Colors:** Dominant palette pastel beige, white, soft yellow. NO red or bright pink.  
- **Details:** Fresh vegetables on table (broccoli, zucchini) for natural ingredients.  
- **Mood:** Safe, calm, happy, healthy.  
- **Negative prompt:** cartoon, illustration, low quality, red clothes, pink background, sad baby, text overlay.

Сценарии в `baby_food_overlay.py` следуют этой стратегии и по структуре соответствуют сценариям ломбарда «объект в углу + градиент» (баночки, бутылочки, ложка, каша и т.д.).
